# 手势控制赛车游戏 - README

## 项目简介

这是一款基于Three.js和MediaPipe的手势控制赛车游戏。玩家可以通过手势控制赛车的转向和加速，体验沉浸式的驾驶乐趣。游戏使用摄像头捕捉手部动作，通过识别左手的位置控制方向，右手的握拳状态和距离控制速度，提供直观的体感交互体验。

## 技术栈

- HTML5 + CSS3
- JavaScript
- Three.js (r128) - 用于3D场景渲染
- MediaPipe Hands - 用于手部关键点检测和手势识别

## 游戏玩法

1. **控制方式**：
   - 左手：左右移动食指控制赛车转向
   - 右手：握拳状态控制前进，手离摄像头越近，车速越快
   - 可点击"左右手逻辑互换"按钮切换左右手控制逻辑

2. **游戏目标**：
   - 驾驶赛车沿赛道行驶，避开障碍物
   - 尽可能快地到达终点线（1000米距离）
   - 屏幕顶部进度条显示完成进度

3. **界面元素**：
   - 右上角：摄像头实时画面（镜像显示）
   - 左下角：游戏数据面板（耗时、剩余距离、控制状态）
   - 底部中央：速度仪表盘
   - 顶部：进度条
   - 终点线：到达时显示"FINISH!"提示并触发烟花效果

## 功能特点

- 真实的3D赛道和赛车模型
- 基于手势的直观控制方式
- 障碍物碰撞检测与物理反馈
- 速度感视觉效果（速度越快，画面抖动越明显）
- 终点线烟花庆祝效果
- 响应式设计，适配不同屏幕尺寸

## 如何运行

1. 将HTML文件在现代浏览器中打开（推荐Chrome或Firefox）
2. 首次运行时，允许浏览器访问摄像头权限
3. 等待模型加载完成（通常需要几秒钟）
4. 使用手势开始游戏

## 注意事项

- 确保在光线充足的环境下玩游戏，以获得更好的手势识别效果
- 保持手部在摄像头可见范围内
- 若手势识别不准确，尝试调整与摄像头的距离（建议50-100cm）
- 遇到性能问题时，可尝试关闭其他浏览器标签页

## 开发说明

游戏核心逻辑分为三个部分：

1. Three.js 3D场景渲染（赛道、赛车、障碍物、灯光等）
2. MediaPipe手势识别与解析（左手转向控制、右手速度控制）
3. 游戏状态管理（速度、位置、碰撞检测、计时等）

主要配置参数可在代码顶部的游戏配置区域调整，包括赛道长度、车道宽度、障碍物数量和最大速度等。
